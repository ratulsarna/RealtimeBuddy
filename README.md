# RealtimeBuddy

The project is now structured as a small monorepo.

- web app: [`apps/web`](./apps/web)
- future mobile app: [`apps/mobile`](./apps/mobile)
- future shared packages: [`packages/shared`](./packages/shared)

Quick start:

```bash
cp apps/web/.env.example apps/web/.env.local
pnpm install
pnpm dev
```
